# Study ledger — template (you own this)

This is the **human-owned** memory the `study-buddy` skill reads before a
session and proposes updates to after one — the same idea as the
data-analyst's semantic layer and the meeting-notes glossary. The product
ships this *template*; your filled-in copy lives in **your** space and is
treated as **your data** — a data reset clears it, the shipped template does
not.

Why human-owned: what you're studying, what you keep getting wrong and when
you last reviewed it are facts about *you*. The skill proposes row updates in
its session receipt; **you** keep the file. Watching intervals stretch from
1 day to 30 and old mistakes stop resurfacing is the honest progress metric —
no percentage needs to be invented.

Ratings use the spaced-repetition convention: **again** (blank), **hard**
(barely), **good** (solid), **easy** (instant). The default ladder is
1 → 3 → 7 → 14 → 30 days; "again/hard" moves an item back down, "easy" climbs
faster.
